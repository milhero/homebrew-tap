# homebrew-tap

My [Homebrew](https://brew.sh) tap. Homebrew resolves `milhero/tap` to this repository — the `homebrew-` prefix as a standalone repo is required by Homebrew's naming convention, which is why this lives on its own.

## Install

```sh
brew install milhero/tap/shotclip
```

## Casks

| Cask | What it does |
| --- | --- |
| [shotclip](https://github.com/milhero/ShotClip) | Save Mac screenshots to a file and the clipboard at the same time |
